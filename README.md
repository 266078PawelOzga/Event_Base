## Running the project

This project is managed using [poetry](https://python-poetry.org/).

```bash
# Install poetry: https://python-poetry.org/docs/
# curl -sSL https://install.python-poetry.org | python3 -

# Activate the environment
poetry env activate

# Install dependencies
poetry install

# Run a script
poetry run hello
```
